1. **Libraries** - _Contains folders named after libraries. In folder:_

- Info file containing version of library used, its name and author _(github link optionally)_
- Description (.md) file with easy to read documentation of library and useful examples
- Src folder containing source files of library (Optionally - only if the library doesn't exist in libraries module in Arduino IDE 2)

2. **Knowledge** - _Information needed for project (calculation, communication protocol etc.)_

- HTML files with downloaded websites
- .md files describing concept
- diagrams and other useful graphics
- folders for each concept (contains above files)

3. **Code** - _Documented code used in project_

4. **Materials** - _things other than code used in project_

- 3D models for printing
- Circuit diagrams
- Components used (offers - may be out of date - or datasheets)

5. **Progress** - _photos and videos showing done elements_
